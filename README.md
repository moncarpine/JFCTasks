# JFC Contacts & Tasks (iOS)

Repository for JFC's Applicant Coding Challenge (iOS).


## Requirements

Welcome to our coding challenge! The task is to develop a contact and task management application.

For iOS Developer Challenge, we are looking for a solution that meets the following requirements:

* iOS: SwiftUI
* Database: Core Data
* Authentication: JWT

The application should provide the following functionalities:

* Full CRUD operations for Businesses and People
* Businesses can belong to one or more Categories
* People may belong to one Business
* Both People and Businesses can have Tags assigned to them
* Interface to manage Tags and Categories
* Both People and Businesses can have Tasks assigned to them
* View to display all the assigned tasks
* All users should have access to all Businesses and People
